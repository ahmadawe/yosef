# شركة النظافة — صفحة واحدة

مشروع صفحة واحدة (One Page) لشركة نظافة — تصميم فخم، متجاوب، ويدعم الوضع الفاتح والداكن.

تشغيل محلي سريع:

1. افتح المجلد `cleaning-company-site`
2. افتح `index.html` في المتصفح (أو استخدم live server في محررك)

الملفات الأساسية:

- `index.html` — الصفحة الرئيسية (RTL، عربية)
- `styles.css` — أنماط التصميم والـ Dark Mode
- `script.js` — تبديل الوضع، smooth scroll، carousel بسيط، معالجة نموذج وهمية

ملاحظات:

- الصور المستخدمة روابط CDN (Unsplash و avatars) — يمكن استبدالها بصور محلية حسب الرغبة.
- النموذج في `index.html` يقوم بمحاكاة الإرسال فقط ولا يرسل بيانات فعلية.

هل تريد أن أقوم بإضافة نسخة من الصور محليًا أو ربط النموذج بخدمة إرسال بريد (مثل Formspree أو API خاص)؟
